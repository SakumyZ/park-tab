# AGENTS.md

## 项目说明

Park Tab 是一个浏览器标签页管理插件，目标是复刻 OneTab 的核心体验，并补足多设备同步、WebDAV 同步、完整导入导出、分组状态保留等能力。

后续开发请优先参考需求文档：

- [docs/requirements.md](docs/requirements.md)

## 协作提示

- 项目核心目标是让不同设备上的标签保存状态保持一致。
- 导入导出必须保留分组、排序、锁定、归档、折叠、备注等完整状态。
- 这是浏览器插件项目，开发时要尽量控制包体积，并优先保证运行性能。
- UI 技术栈使用 Svelte + Tailwind CSS，首版只考虑 Chromium 系浏览器。
- 开发时应避免破坏已有数据模型的可迁移性，涉及 schema 变更时需要同步更新需求或设计文档。
- 新功能追加以及功能开发完成需要更新需求文档
